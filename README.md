# Requirements

* Xcode
* MongoDB
* Node

# How to build

1) Clone the repository

2) Go to directory

```bash
$ cd Twitter-API
```

3) Open Project in Xcode

4) Run The Server From Twitter-API Folder

```bash
node index.js
```

5) Compile and run the app in your simulator
